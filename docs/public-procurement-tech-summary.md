# 공공조달 기술 요약서

> **제목:** AX 공공임대주택 AI 시설물 안전유지관리 플랫폼  
> **사업명:** AX-SPRINT — 국가R&D 지능형 공공임대주택 유지관리 자동화  
> **작성일:** 2026년 4월  
> **분류:** 공공조달 기술 설명 자료 (비공개)

---

## 1. 제품 개요

AX 플랫폼은 공공임대주택의 **시설물 안전관리, 민원 처리, 행정 자동화**를 AI와 IoT로 통합한 **클라우드 기반 SaaS 솔루션**입니다.

**핵심 차별점:**
1. **드론 AI 점검 자동화** — 외벽 균열/결함 자동 탐지 (Mask R-CNN / Y-MaskNet)
2. **설명가능 AI(XAI)** — 모든 AI 판단에 근거·신뢰도 제공, 감사 추적 가능
3. **Human-in-the-Loop** — AI 결과는 항상 초안, 담당자 검토 후 확정 (책임 소재 명확)
4. **통합 자동화** — 민원·점검·계약·알림 업무 80~100% 자동화
5. **멀티테넌트** — 기관별 데이터 완전 격리, 역할 기반 접근제어

## 2. 기술 스택

| 구분 | 기술 | 용도 |
|------|------|------|
| 백엔드 | NestJS (TypeScript) | REST API, 비즈니스 로직 |
| 데이터베이스 | CouchDB 3.3 | 문서형 DB, 멀티테넌트 격리 |
| 캐시/큐 | Redis 7 + Bull | 비동기 잡 큐, AI 처리 |
| 파일 저장 | MinIO (S3 호환) | 이미지·영상·보고서 |
| AI/Vision | Mask R-CNN, Y-MaskNet, OpenCV WASM | 결함 탐지, 균열 측정 |
| AI/NLP | Claude API (Anthropic) | 진단 의견, 민원 분류 |
| 프론트엔드 | Angular 18 (Standalone) | 관리자 웹 |
| 모바일 | Ionic + Angular | 점검자 앱 (오프라인 우선) |
| 인프라 | Docker Compose / 클라우드 배포 가능 | 온프레미스·클라우드 모두 지원 |

## 3. 주요 기능 목록

### 3.1 AI 자동 점검 (Phase 2)

| 기능 | 설명 | 자동화 단계 |
|------|------|-------------|
| 드론 미션 관리 | 비행 계획 수립, 영상 업로드, AI 분석 트리거 | 반자동 |
| AI 결함 탐지 | 이미지에서 균열·누수·박리 자동 탐지, 신뢰도 분류 | 탐지 자동 / 확정 수동 |
| 균열 정밀 측정 | 균열 폭(mm) 자동 측정, KCS 기준 연계 | 자동 측정 |
| AI 진단 의견 | 결함 데이터 기반 기술 의견 초안 자동 생성 | 생성 자동 / 승인 수동 |
| 결함 이력 관리 | 위치·심각도·이미지·작업지시 통합 관리 | 통합 관리 |

### 3.2 민원 자동화

| 기능 | 설명 | 자동화율 |
|------|------|----------|
| AI 민원 분류 | 접수 즉시 카테고리·긴급도·배정팀 자동 추천 | 추천 자동 / 확정 수동 |
| 상태 자동화 | 민원 상태 변경 시 담당자 알림 자동 발송 | 100% |
| SLA 관리 | 카테고리별 처리 기한 자동 계산·추적 | 100% |

### 3.3 IoT 센서 연동

| 기능 | 설명 |
|------|------|
| 7종 센서 지원 | 온도·습도·진동·누수·CO2·압력·수위 |
| 실시간 경보 | 임계치 초과 시 즉시 경보 생성 |
| 데이터 이력 | 시계열 데이터 저장 및 이상 감지 |

### 3.4 위험도 예측·예지정비

| 기능 | 설명 |
|------|------|
| 위험도 계산 | 결함(30%)+균열(25%)+센서(20%)+민원(15%)+노후도(10%) 가중 합산 |
| 유지보수 권장 | 위험도 기반 자동 정비 계획 생성 |
| 장기수선 지원 | 예산·공사 규모·우선순위 시각화 |

### 3.5 행정 자동화 (RPA)

| 업무 | 자동화율 | 내용 |
|------|----------|------|
| 관리비 고지서 | 80% | 자동 생성 + 담당자 최종 확인 |
| 계약 만료 알림 | 100% | 30일/7일 전 자동 발송 |
| 민원 접수·분류 | 70% | AI 분류 + 담당자 확정 |
| 점검 일정 생성 | 90% | 자동 스케줄링 + 담당자 승인 |

## 7. 기대 효과 요약

| 효과 항목 | 기대 수치 | 측정 방법 |
