## 1. 시험 개요

### 1.1 시험 목적

본 시험은 에이톰-AX(AX 기반 공공임대주택 시설물 안전·유지관리 최적화 및 지능형 행정 자동화 솔루션)의 기능 완전성, 성능 적정성, 보안 강건성, 신뢰성을 검증하기 위하여 실시한다.  
본 시험은 국토교통부 AI 응용제품 신속 상용화 지원사업(AX-SPRINT) 신청에 요구되는 기술성숙도(TRL 8) 달성을 자체 증빙하기 위한 목적으로 수행하며, v1.3은 TRL-8 2차 기능 고도화(외부 연동, AI 시각화, 법정 보고서, 신규 결함 유형, 시스템 로그, k6 부하 테스트)를 반영하여 갱신한 성적서이다.

### 1.2 시험 대상

| 구분 | 대상 | 버전 | 주요 변경 사항 (v1.2 → v1.3) |
|------|------|------|------|

| 백엔드 API | AX API Server (NestJS 10) | v0.1.0 | 외부 연동 모듈(KALIS-FMS·세움터), 법정 보고서 생성, /static/data 정적 파일 서빙 |
| AI 워커 | AX AI Worker | v1.0.0 | DefectType 12종 확장 (DRYVIT·FIRE_RISK_CLADDING 추가) |
