# UAT 체크리스트 — integ-xAI-PublicHousing

> 에이톰-AX | AX 기반 공공임대주택 시설물 안전유지관리 플랫폼  
> 평가자·시연자가 수행하는 기능 검증 체크리스트 (최종 통합본 기준)  
> 최종 갱신: 2026-04-15

---

## 사전 준비

```bash
# 1. 인프라 기동
docker compose -f docker-compose.dev.yml up -d

# 2. 의존성 + 빌드
yarn install && yarn build:shared

# 3. Seed 데이터 투입
yarn workspace @ax/api ts-node src/database/seed.ts
yarn workspace @ax/api ts-node src/database/seeds/demo.seed.ts

# 4. 앱 실행 (터미널 2개)
yarn dev:api          # http://localhost:3000
yarn dev:admin        # http://localhost:4200
```
