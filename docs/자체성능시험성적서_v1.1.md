## 1. 시험 개요

### 1.1 시험 목적

본 시험은 에이톰-AX(AX 기반 공공임대주택 시설물 안전·유지관리 최적화 및 지능형 행정 자동화 솔루션)의 기능 완전성, 성능 적정성, 보안 강건성, 신뢰성을 검증하기 위하여 실시한다.  
본 시험은 국토교통부 AI 응용제품 신속 상용화 지원사업(AX-SPRINT) 신청에 요구되는 기술성숙도(TRL 8) 달성을 자체 증빙하기 위한 목적으로 수행하며, 시험 결과는 소프트웨어 품질 인증, 공공 조달 납품, 감리 등에 활용한다.

### 1.2 시험 대상

| 구분 | 대상 | 버전 |
|------|------|------|
| 프론트엔드 | AX Admin Web (Angular 18) | v1.0.0 |
| 백엔드 API | AX API Server (NestJS 10) | v1.0.0 |
| AI 워커 | AX AI Worker | v1.0.0 |
| 배경 작업 | AX Job Worker | v1.0.0 |

## 4. 기능 시험 결과

> **시험 방법:** 각 시나리오를 역할별 사용자 계정으로 순서대로 실행하고 예상 결과와 비교한다.  
> **결과 기재:** 시험자가 실제 수행 후 결과, 판정, 비고를 기재한다.

### B.2 k6 부하 테스트 스크립트 (대시보드)

```javascript
// k6 run load_test.js
import http from 'k6/http';
import { check, sleep } from 'k6';

export const options = {
  vus: 50,       // 50 가상 사용자
  duration: '60s',
};

const TOKEN = __ENV.TOKEN;

export default function () {
  const res = http.get('http://localhost:3000/api/v1/dashboard', {
    headers: { Authorization: `Bearer ${TOKEN}` },
  });

  check(res, {
    'status 200': (r) => r.status === 200,
    '응답 500ms 이내': (r) => r.timings.duration < 500,
  });

  sleep(1);
}
```

### B.3 보안 헤더 확인

```bash
# HTTP 보안 헤더 목록 확인
curl -I http://localhost:3000/api/v1/health | grep -E "X-Frame|X-Content|Strict-Transport|Content-Security"

# Rate Limit 테스트 (101번째 요청에서 429 확인)
for i in $(seq 1 105); do
  STATUS=$(curl -s -o /dev/null -w "%{http_code}" \
    -H "Authorization: Bearer $TOKEN" \
    http://localhost:3000/api/v1/users)
  echo "Request $i: $STATUS"
done
```
